# Documentación de Pakal

Esta carpeta contiene la documentación de arquitectura y ejemplos de uso del motor Pakal.

Archivos incluidos:
- `ARCHITECTURE.md` — Visión general y flujo de ejecución.
- `SUBSYSTEMS.md` — Documentación por subsistema con rutas a archivos relevantes.
- `USAGE.md` — Guía mínima para compilar y ejecutar, ejemplo `main.cpp`.
- `EXAMPLES.md` — Ejemplos prácticos con snippets.
